# 순간일기 (Moment Diary) — 음성+카메라 일기 앱

목소리로 오늘의 느낌을 말하면 글이 되고, 카메라로 지금 보고 있는 날씨·풍경을 담는 **책 모양 일기장**.
월말에는 한 달의 기록을 이어 붙여 **짧은 브이로그 영상**으로 만들어 줍니다.

> 스토어 판매를 목표로 하는 앱입니다. 현재는 핵심 구성(기능 골격)을 다지는 단계이며, 디자인은 추후 교체 예정입니다.

## 주요 기능

| 기능 | 설명 |
|---|---|
| 🎙️ 음성 일기 | 마이크 버튼을 누르고 말하면 실시간으로 텍스트 변환 (Web Speech API). 미지원 환경에서는 직접 입력 안내로 자동 전환 |
| 📷 카메라 기록 | **사진** 또는 **2초 짧은 영상** 중 선택해 촬영 |
| 🎞️ 필터 | 빈티지(세피아+비네트+필름 그레인) · 색감 강조 · 볼록거울(어안) — 촬영 전 실시간 미리보기 |
| 📖 책 UI | 다이어리 속지에 미디어(위) + 텍스트 일기(아래), 페이지 넘김 애니메이션, 날씨 이모지 |
| 🎬 월말 브이로그 | 한 달치 사진·영상을 자동 합성해 "1일 — 31일의 여정" webm 영상 생성 + 다운로드 |
| 💾 저장 | 전부 기기 안 IndexedDB에 저장 (서버 전송 없음). 개별 삭제 · 전체 초기화 지원 |

## 실행 방법

빌드 도구가 필요 없는 순수 정적 웹앱입니다. `index.html`을 브라우저(크롬 권장)로 열면 됩니다.
카메라·마이크 권한이 필요하며, 배포 시에는 https(GitHub Pages 등)에서 열어야 권한이 동작합니다.

## 테스트

기능을 추가/변경할 때마다 스모크 테스트를 실행합니다 (playwright 전역 설치 + Chromium 필요):

```bash
NODE_PATH=$(npm root -g) PLAYWRIGHT_BROWSERS_PATH=/opt/pw-browsers node tests/smoke.js
```

가짜 카메라 장치로 촬영 → 저장 → 넘김 → 브이로그 합성 → 새로고침 유지 → 삭제까지 전 구간을 검증합니다.

## 기술 구성

- 무의존성: HTML + CSS + JS 3개 파일, 외부 라이브러리 없음
- 카메라: `getUserMedia` → Canvas 실시간 필터 → 사진(`toBlob`) / 영상(`captureStream`+`MediaRecorder`)
- 음성: `SpeechRecognition`(ko-KR) + 실패 시 직접 입력 안전망
- 브이로그: Canvas 슬라이드쇼를 `MediaRecorder`로 녹화해 webm 생성
- 저장: IndexedDB `moment-diary`(미디어 Blob 포함), 작은 상태는 localStorage(`momentDiary:` 접두사)
