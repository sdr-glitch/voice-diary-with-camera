# 순간일기 (Moment Diary) — 음성+카메라 일기 앱

목소리로 오늘의 느낌을 말하면 글이 되고, 카메라로 지금 보고 있는 날씨·풍경을 담는 **책 모양 일기장**.
월말에는 한 달의 기록을 이어 붙여 **짧은 브이로그 영상**으로 만들어 줍니다.

> 스토어 판매를 목표로 하는 앱입니다. 현재는 핵심 구성(기능 골격)을 다지는 단계이며, 디자인은 추후 교체 예정입니다.

## 주요 기능

| 기능 | 설명 |
|---|---|
| 🎙️ 음성 일기 | 마이크 버튼을 누르고 말하면 실시간으로 텍스트 변환 (Web Speech API). 미지원 환경에서는 직접 입력 안내로 자동 전환 |
| 📷 카메라 기록 | **사진** 또는 **2초 짧은 영상** 중 선택해 촬영 |
| 🎞️ 필터 | 빈티지(세피아+비네트+필름 그레인) · 색감 강조 · 볼록거울(어안) — 촬영 전 실시간 미리보기 |
| 📖 책 UI | 다이어리 속지에 미디어(위) + 텍스트 일기(아래), 페이지 넘김 애니메이션, 날씨 이모지 |
| 🎀 스티커 꾸미기 | 촬영한 사진·영상 위에 이모지 스티커를 붙이고 드래그로 배치, 크기 조절 (인스타 스토리 방식) |
| 📅 사진 달력 홈 | 앱을 열면 달력이 홈. 사진 있는 날은 **셀에 썸네일**(영상은 첫 장면 정지컷), 하루 여러 장이면 **대표 사진 선택**. 날짜를 누르면 **바로 아래에서 크게 보고 사진·글을 그 자리에서 수정**(별도 페이지 없음) |
| 🖼️ 사진 꾸미기 | 폴라로이드 **프레임 색(아이보리·블랙 등)**과 사진 위 **마스킹 테이프 색**을 설정에서 골라 적용 |
| 🖼️ 갤러리 가져오기 | 기기 갤러리의 사진·영상을 불러와 채움. **가져온 사진에도 필터가 적용**되고, **손가락으로 모으고 벌려 크기·위치를 맞춰**(핀치 줌) 잘리지 않게 조절 |
| 🔲 모아보기 | 모든 기록을 격자 갤러리로 한눈에. **길게 누르면 흔들림(편집), 위아래로 쓸면 사진이 와르르** 쏟아지는 애니메이션 |
| 📊 기분 통계 | 한 달 기분 분포 막대그래프 + 최다 기분에 맞춘 **공감·위로·응원 카드**. 기분을 누르면 **그 기분의 일기만 모아보기** |
| 📼 브이로그 보관함 | 만든 브이로그를 **바둑판(그리드)**으로 모아 보고, 타일을 누르면 **큰 화면**으로 재생·저장·삭제. 영상은 **mp4(H.264)**로 저장돼 아이폰 사진앱·파일·카카오톡 어디서나 재생됨 |
| 🖼️ 다이어리 표지 사진 | 일기장(책)마다 **원하는 사진을 표지로** 설정 (설정 → 내 일기장 → 표지 사진). 책장에서 사진 표지로 보임 |
| 🗓️ 일정·기념일 | 달력의 **모든 날짜(지난 날·오늘·미래, 1~12월 어느 달이든)**를 눌러 **일정·기념일을 저장**(기념일은 매년 반복). 일정 있는 날은 달력에 표시되고, **다가오는 일정**(오늘·내일·그 다음)을 D-day로 한눈에 |
| ⏰ 기록 알림 | 매일 원하는 시간에, 아직 기록이 없으면 배너·알림으로 알려줌 |
| 📚 여러 일기장 (책장) | 첫 화면이 **책장** — 일상 일기장, 취미 일기장, 여행 일기장처럼 **주제별로 여러 권을 따로** 만들어 써요. 책을 눌러 열고, '＋'로 새로 만들고, 설정에서 이름 변경·삭제. 일기장마다 기록·작성자·대표사진이 **완전히 분리** |
| 🏷️ 일기장 주제 | 일상·아기성장·부부/커플·반려동물·연애·취미·러닝/운동·식물집사·여행·공부·맛집·자유 중 선택, **직접 주제 만들기**도 가능 (새 일기장 만들 때 + 설정) |
| 👥 함께 쓰기 | 가족·연인을 작성자로 추가해 한 일기장을 같이 씀 (기록마다 작성자 표시) |
| 🔁 교환일기 | 내 일기를 파일로 **내보내기**, 받은 파일을 **가져와 합치기**(중복 자동 제외). 실시간 클라우드 동기화는 추후 서버 도입 예정 |
| 🙂 기분 기록 | 사진·글과 함께 그날의 기분(설렘·행복·평온·지침 등)을 남김 |
| 🎞️ 필터 8종 | 필터 없음·빈티지·색감 강조·볼록거울·필름·폴더폰(레트로)·흑백·몽환 |
| 🎥 길이 자유 영상 | 셔터로 녹화 시작/정지 — 원하는 만큼 길게(고정 길이 아님) 촬영 |
| 🎬 월말 브이로그 + 편집 | **폴라로이드 프레임** 안에서 움직이는 사진(켄번즈)·영상(**사진·영상만, 글 제외**). 목표 길이(짧게/보통/길게)에 압축, 넣을 장면·순서·제목·**마무리 한마디** 편집. 배경음악 4종. **만드는 동안 다른 화면으로 옮겨 새 일기를 써도 계속 만들어지고**(진행 칩 표시), 완성되면 **자동으로 보관함에 저장 + 알림** |
| 🔊 소리 | 버튼(비눗방울)·촬영·종이 넘기는 소리·배경음악 전부 자체 합성. 설정에서 켜기/끄기·음량 조절 |
| 🎵 배경음악·효과음 | 기본 음원(피아노·오르골)과 전환 효과음은 **전부 앱이 웹오디오로 직접 합성** — 제3자 저작물이 없어 상업용 사용에 제한 없음. 내 음원 파일 추가도 지원 |
| 💾 저장 | 전부 기기 안 IndexedDB에 저장 (서버 전송 없음). 개별 삭제 · 전체 초기화 지원 |

## 실행 방법

빌드 도구가 필요 없는 순수 정적 웹앱입니다. `index.html`을 브라우저(크롬 권장)로 열면 됩니다.
카메라·마이크 권한이 필요하며, 배포 시에는 https(GitHub Pages 등)에서 열어야 권한이 동작합니다.

## 테스트

기능을 추가/변경할 때마다 스모크 테스트를 실행합니다 (playwright 전역 설치 + Chromium 필요):

```bash
NODE_PATH=$(npm root -g) PLAYWRIGHT_BROWSERS_PATH=/opt/pw-browsers node tests/smoke.js
```

가짜 카메라 장치로 촬영 → 저장 → 넘김 → 브이로그 합성 → 새로고침 유지 → 삭제까지 전 구간을 검증합니다.

## 기술 구성

- 무의존성: HTML + CSS + JS 3개 파일, 외부 라이브러리 없음
- 카메라: `getUserMedia` → Canvas 실시간 필터 → 영상(기본, `captureStream`+`MediaRecorder`) / 사진(`toBlob`). 영상 중심 + 아래 짧은 메모 구성
- 음성: `SpeechRecognition`(ko-KR) + 실패 시 직접 입력 안전망
- 브이로그: Canvas 슬라이드쇼를 `MediaRecorder`로 녹화해 webm 생성, 소리는 WebAudio 합성 트랙을 합침
- 음원 라이선스 정책: 외부 무료 음원은 재배포 시 라이선스 위반 위험이 있어 **기본 소리는 전부 자체 합성**(저작권 완전 안전). 사용자 업로드 음원은 사용자 책임이며 UI에서 이용 범위 확인을 안내
- 저장: IndexedDB `moment-diary`(미디어 Blob 포함), 작은 상태는 localStorage(`momentDiary:` 접두사)
